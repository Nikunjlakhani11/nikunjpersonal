# E-Commerce Soap Website Project

## Overview
This document provides comprehensive documentation for the E-Commerce Soap Website project. The goal of this project is to create a platform for users to browse, order, and manage soap products online.

## Features
- **User Registration and Login**: Allows users to create accounts and log in to manage their orders.
- **Product Catalog**: A comprehensive catalog where users can view all available soap products.
- **Shopping Cart**: Users can add products to a shopping cart and review their selections before checkout.
- **Order Management**: Users can view their order history and track the status of their recent purchases.
- **Payment Integration**: Secure payment processing for a seamless checkout experience.
- **Admin Dashboard**: Admins can manage product listings, track sales, and handle user accounts.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript (Framework: React.js)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **SOAP Services**: Utilized for specific functionalities related to product management.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Nikunjlakhani11/nikunjpersonal.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nikunjpersonal
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and visit `http://localhost:3000`.

## Usage
- Users can register, log in, browse products, and manage their shopping cart.
- Admins can log into the admin dashboard to manage product listings and user accounts.
